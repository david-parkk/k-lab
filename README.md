# CHALLIFY

### 1. 프로젝트 설명

##### 1-1. 프로젝트 기획

 > 첼린지 정보제공 웹 서비스

##### 1-2. 프로젝트 목적
 - REST API에 대한 전반적인 이해
 - git을 이용한 버젼관리
 - 쿠키와 같은 프론트사이드 데이터에 대한 이해
 
### 2. Chalify 기능
  - 첼린지 정보제공
  - 나이에 따른 위험성 첼린지 필터(20세이상)
  - 계시판 및 댓글
  - 회원가입 로그인
  
### 3. 아키텍처
**기술 스택**
* Vue.js
* Vuex
* Vue CLI
* AWS S3
